# TradingBees Journal

A free & open-source MT5 trading journal built for Forex traders.

## Features
- 📊 PnL Dashboard
- 📈 Equity Curve
- 📉 Drawdown Analytics
- 📒 Trading Journal
- 😊 Psychology Tracking
- 📷 Trade Screenshots
- 📥 MT5 CSV Import
- 📄 MT5 HTML Import
